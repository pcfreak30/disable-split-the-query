Simple WordPress plugin to always disable split_the_query

== Description ==

This plugin will disable the split_the_query feature in WP_Query that can hinder performance in some cases

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/disable-split-the-query` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
4. Test!
